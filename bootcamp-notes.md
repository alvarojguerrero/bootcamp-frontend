### __Comandos Terminal__
#### ls -l
list dir contents long format

#### ls -a
list dir contents including hidden

#### .carpeta
oculta(ex: carpetas, archivos)

#### mkdir nombre
crear carpeta

#### rm -r dir
remove dir

#### rm -rf *

#### cp -r dir1 dir2
copiar (dir) y se va a llamar (dir2)

#### cd -
devolver al anterior directorio

#### mv file1 file2
renombrar un archivo

#### touch.extension
crear archivo

#### more file
mirar archivo
